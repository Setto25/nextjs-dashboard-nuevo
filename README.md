# 👁️ Plataforma Web de Capacitación Clínica – UCIN

Plataforma web diseñada para centralizar contenidos de formación, protocolos, manuales de equipos y biblioteca digital en una unidad de cuidados intensivos neonatales (UCIN). Creada para mejorar el acceso al conocimiento, fomentar la autogestión del equipo clínico y digitalizar procesos de aprendizaje.

## 🌟 Propósito

* Asegurar el acceso oportuno a contenidos clínicos clave.
* Reducir la dependencia del correo y la memoria institucional.
* Facilitar la autogestión de capacitaciones y contenido por parte del equipo.

## 🌐 Tecnologías utilizadas

* **Next.js** (framework React para SSR).
* **Prisma ORM** + **SQLite** (modelo de datos local).
* **Autenticación con cookies** y permisos por rol.
* **CSS/HTML** para estilización.
* Hospedaje en entorno local o servidor institucional.

## 🧰 Módulos principales

* **Capacitaciones clínicas** (agregables por los usuarios con permiso de administrador).
* **Sección de protocolos neonatales**.
* **Biblioteca digital** (PDFs, manuales, guías).
* **Gestón de usuarios** con roles diferenciados.
* **Módulo de autogestión** para crear secciones, contenidos y usuarios sin intervención del desarrollador.
